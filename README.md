# Pastebin Lite

A simple Pastebin-like web application built with Next.js and Prisma.

## Features
- Create and store text pastes
- Generate a shareable link
- View pastes using the link
- Optional expiration by time or number of views

## Tech Stack
- Next.js (App Router)
- Prisma ORM
- PostgreSQL (Neon)
- Node.js

## Setup Instructions

```bash
npm install
npm run dev
